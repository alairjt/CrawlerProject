# CrawlerProject
The Crawler Project

## Run projects

### Run API and Web with Docker Compose 
    - `docker-compose up`

#### API
    - http://localhost:5000/deputies

#### Web
    - http://localhost:3000

### Run Crawler
    - `git clone git@github.com:alairjt/CrawlerNode.git`
    - `npm install -g codecov`
    - `npm install`
    - `npm start`
